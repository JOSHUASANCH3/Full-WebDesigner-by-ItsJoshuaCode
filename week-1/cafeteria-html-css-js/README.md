- **Cafeteria Express** es una landing page moderna para una hamburguesería ficticia, desarrollada con HTML, CSS y JavaScript.  
  - Esta pagina principal posee una estructura semantica en el HTML :
    - Header
      - Hero
      - Navbar
    - Seccion de Banner
    - Main

      - Top Categories

        - Card Blogs
    - Footer
      - Menu

        - Information
